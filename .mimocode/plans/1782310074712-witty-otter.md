## Phase 1: Security Hardening

### 1.1 — OTP: `Math.random()` → `crypto.randomInt`
**File:** `backend/src/lib/tokens.js:52-54`
- Add `import { randomInt } from 'node:crypto'` (already imports from `node:crypto` on line 1)
- Replace `Math.floor(100000 + Math.random() * 900000)` with `randomInt(100000, 1000000)`

### 1.2 — Content-Disposition header injection fix
**New file:** `backend/src/lib/sanitize.js` — `safeFilename(name)` function that strips unsafe chars
**Edit:** `backend/src/routes/files/index.js:88` — wrap `row.originalName` with `safeFilename()`
**Edit:** `backend/src/routes/corporate/index.js:608` — wrap `fileName` with `safeFilename()`

### 1.3 — Add `APP_URL` to production validation
**File:** `backend/src/config/env.js:33`

## Phase 2: Backend Resilience

### 2.1 — Graceful shutdown
**File:** `backend/src/index.js` — rewrite to:
- Store server reference from `app.listen()`
- Store interval handle from `startComplianceScheduler()`
- Add `SIGTERM`/`SIGINT` handlers that: stop server, clear interval, call `closeDb()`, exit after 10s timeout
- Import `closeDb` from `./db/index.js`

### 2.2 — Scheduler stop function
**File:** `backend/src/services/compliance/scheduler.js`
- Add `export function stopComplianceScheduler(handle) { if (handle) clearInterval(handle) }`

### 2.3 — Deep health check + move outside rate limiter
**File:** `backend/src/server.js` — restructure:
- Import `sqlite` from `./db/index.js`
- Add health check route `app.get('/api/health', ...)` BEFORE `app.use('/api', apiRateLimit)`
- Health check runs `sqlite.prepare('SELECT 1').get()`, returns 200 or 503
- Remove health check from `backend/src/routes/public.js:58`

### 2.4 — Configurable trust proxy
**File:** `backend/src/config/env.js` — add `TRUST_PROXY: z.coerce.number().default(1)` to schema
**File:** `backend/src/server.js:13` — change to `app.set('trust proxy', env.TRUST_PROXY)`

### 2.5 — CORS preflight caching
**File:** `backend/src/server.js:15` — add `maxAge: 86400` to cors options

### 2.6 — Guard seed against production
**File:** `backend/src/db/seed.js` — add production check at top of `runSeed()`:
```js
if (env.NODE_ENV === 'production') throw new Error('Cannot seed in production')
```
Import `env` from `../config/env.js`

### 3.1 — Lazy loading all routes
**File:** `frontend/src/App.jsx`
- Convert all 65+ route imports to `React.lazy(() => import(...))` 
- Keep layout imports (PublicLayout, AuthLayout, etc.) eager
- Add `<Suspense>` wrapper with a loading spinner around `<Routes>`

### 3.2 — Vite build optimizations
**File:** `frontend/vite.config.js`
- Add `build.sourcemap: false`
- Add `build.rollupOptions.output.manualChunks` splitting vendor (react, react-dom, react-router-dom), charts (recharts), forms (react-hook-form, zod)

**File:** `frontend/index.html`
- Add `<meta name="description">`, OG tags, Twitter Card tags, `<meta name="theme-color">`
